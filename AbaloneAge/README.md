# AbaloneAge

Du an mon hoc Machine Learning cho bai toan du doan tuoi cua abalone.

## Cau truc chinh

- `docs/`: de bai, tai lieu tham khao, ghi chu hop nhom.
- `data/`: du lieu goc, du lieu trung gian, du lieu da xu ly.
- `notebooks/`: cac notebook phan tich va thu nghiem.
- `src/`: ma nguon chinh cua du an.
- `experiments/`: thu nghiem theo tung mo hinh.
- `outputs/`: mo hinh da luu, metric, hinh anh, log.
- `reports/`: bao cao, slide, phan cong cong viec.
- `tests/`: test cho data pipeline va model.

## Cach bat dau

1. Tao moi truong ao.
2. Cai thu vien trong `requirements.txt`.
3. Chay notebook `01_kham_pha_du_lieu.ipynb`.
4. Hoan thien cac module trong `src/`.
