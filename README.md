# AbaloneAge

Dự án môn học Machine Learning cho bài toán dự đoán tuổi của abalone.

## Cấu trúc chính

- `docs/`: đề bài, tài liệu tham khảo, ghi chú họp nhóm.
- `data/`: dữ liệu gốc, dữ liệu trung gian, dữ liệu đã xử lý.
- `notebooks/`: các notebook phân tích và thử nghiệm.
- `src/`: mã nguồn chính của dự án.
- `experiments/`: thử nghiệm theo từng mô hình.
- `outputs/`: mô hình đã lưu, metric, hình ảnh, log.
- `reports/`: báo cáo, slide, phân công công việc.
- `tests/`: test cho data pipeline và model.

## Cách bắt đầu

1. Tạo môi trường ảo.
2. Cài thư viện trong `requirements.txt`.
3. Chạy notebook `01_kham_pha_du_lieu.ipynb`.
4. Hoàn thiện các module trong `src/`.
5. Chạy các script trong `experiments/` để lưu kết quả so sánh mô hình.
